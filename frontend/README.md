# 節奏型行為追蹤 Web App

🎯 一個簡單易用的習慣追蹤應用，幫助你建立良好的生活節奏！

## 🌐 線上體驗

**網址**: [https://你的用戶名.github.io/vibe-tracker](https://你的用戶名.github.io/vibe-tracker)

## ✨ 功能特色

- ✅ **記錄每日習慣** - 簡單標記完成狀態
- 📊 **智能統計分析** - 自動計算完成率、連續天數
- 📈 **趨勢追蹤** - 了解你的進步方向
- 💾 **本地儲存** - 資料保存在瀏覽器中
- 📱 **響應式設計** - 手機電腦都好用

## 🚀 快速開始

1. 開啟網頁應用
2. 輸入習慣名稱（如：運動、閱讀、早睡）
3. 選擇日期和完成狀態
4. 點擊查詢查看統計資料

## 📊 統計指標說明

- **完成率**: 在查詢期間內的完成百分比
- **目前連續**: 從最新日期開始的連續完成天數
- **最長連續**: 歷史最佳連續記錄
- **趨勢**: 分析你的習慣執行趨勢（上升/下降/穩定）

## 💡 使用技巧

- 建議每天固定時間記錄
- 可以同時追蹤多個習慣
- 利用備註功能記錄心得
- 定期查看統計數據調整策略

## 🔧 技術架構

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **儲存**: localStorage (瀏覽器本地儲存)
- **部署**: GitHub Pages
- **設計**: 響應式 + 現代化 UI
